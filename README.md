# Trade Suite

Trade Suite is a trader performance platform designed to help discretionary traders build, test, and refine their trading systems.

## 🎯 Purpose

Trade Suite focuses on:

- Playbook development (structured setups and rules)
- Trade journaling
- Behavioral tracking (rules, emotions, mistakes)
- Performance analytics
- Strategy evolution over time

This is **not** a charting or execution platform.

## 🚀 Phase 1 (Current Scope)

- Rule library
- Setup families, variants, and ranks
- Setup versioning
- Trade journaling
- Backtest vs live trade separation
- Behavioral tracking (rules, emotions, mistakes)
- Analytics (setup, version, discipline, execution)

## 🧠 Core Concept

Trade Suite helps traders answer:

1. Which setups actually work?
2. Which changes improved performance?
3. Where do I break my rules?
4. How do emotions affect my results?
5. How much of the move do I actually capture?
